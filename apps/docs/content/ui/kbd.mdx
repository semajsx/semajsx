## Usage

```tsx
import { Kbd } from "@semajsx/ui/components/kbd";

<Kbd>Ctrl</Kbd>

<p>
  Press <Kbd>Cmd</Kbd> + <Kbd>K</Kbd> to open the command palette.
</p>
```
