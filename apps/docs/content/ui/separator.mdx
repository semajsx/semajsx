## Usage

```tsx
import { Separator } from "@semajsx/ui/components/separator";

// Horizontal (default)
<Separator />

// Vertical
<Separator orientation="vertical" />
```

## Props

| Prop          | Type                         | Default        | Description |
| ------------- | ---------------------------- | -------------- | ----------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Orientation |
