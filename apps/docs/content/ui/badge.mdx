## Usage

```tsx
import { Badge } from "@semajsx/ui/components/badge";

<Badge>Default</Badge>
<Badge color="success">Published</Badge>
<Badge color="warning">Draft</Badge>
<Badge color="danger">Deprecated</Badge>
```

## Props

| Prop       | Type                                                                 | Default     | Description   |
| ---------- | -------------------------------------------------------------------- | ----------- | ------------- |
| `color`    | `"default" \| "success" \| "warning" \| "danger" \| "info" \| "tip"` | `"default"` | Badge color   |
| `children` | `JSXNode`                                                            | —           | Badge content |
